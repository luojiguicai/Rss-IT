## 📋 贡献指南

1. Fork 本仓库
2. 编辑 `data/entries.json`，按 schema 添加新条目
3. 运行 `python3 scripts/generate-readme.py` 重新生成 README
4. 运行 `python3 scripts/check-links.py` 验证链接
5. 提交 PR

### 数据格式

每个条目包含 `id`、`name`、`category`、`author`、`desc`、`links`（optional）、`tags`、`quality`（1-5）。
links 中只填真实存在的链接，不要留空值。

### 自动维护

- `scripts/check-links.py` — 检查所有链接可用性
- `scripts/discover-sources.py` — 从 Android Weekly 和 RSS 列表发现新源
